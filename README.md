# node-express-mysql
> 本项目采用node+express+mysql开发服务端，采用前后端分离式，项目不包括前端部分，可运行在浏览器直接查看接口，运行项目前请先导入数据库，然后修改config中数据库信息
## Build Setup
``` bash
# install dependencies
npm install

# serve with hot reload at localhost:3000
npm start
